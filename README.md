## 💡 Introduction

TODO List web application

- Create a task
- Delete a task
- Mark a task as done (check)
- View all tasks statuses

<br>

## 🧪 Technologies and Services

- [React](https://reactjs.org)
- [Vite](https://vitejs.dev/)

<br>

## 🖥 Pre-requisites:

- NodeJS

<br/>

## 🚀 How to Run it?

From project's root file:

```sh
$ npm install
$ npm run dev
```
